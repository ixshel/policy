## Publishing with GitHub Pages

Follow these steps to make the privacy policy publicly accessible via GitHub Pages:

### 1. Push this repository to GitHub

If you haven't already, create a repository on GitHub and push the contents:

```bash
git init
git add .
git commit -m "Add privacy policy page"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

### 2. Enable GitHub Pages

1. Go to your repository on **GitHub.com**.
2. Click **Settings** → **Pages** (in the left sidebar).
3. Under **Source**, select **Deploy from a branch**.
4. Choose the **`main`** branch and the **`/ (root)`** folder, then click **Save**.

### 3. Access your privacy policy

After a few minutes, your privacy policy will be live at:

```
https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/privacy-policy.html
```

Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your actual GitHub username and repository name.

### 4. Update the contact email

Open `privacy-policy.html` and replace every occurrence of `YOUR_EMAIL_HERE` with your real
support/contact email address before publishing.
